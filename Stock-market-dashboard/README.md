 A fully Dynamic Stock Market Analysis Dashboard built in Tableau Public!

📊 What makes this special? The data isn't static — it's extracted LIVE from Yahoo Finance API using Python, transformed with pandas, and fed directly into Tableau for real-time insights.

🔧 Tech Stack:
• Python (yfinance + pandas + numpy) — for API extraction & transformation
• Tableau Public — for interactive visualization
• Automated pipeline — data refreshes automatically

📈 What the dashboard tracks:
• 40+ stocks across US Markets (AAPL, AMZN, NVDA, AMD...) and India NSE (RELIANCE, TCS, HDFC...)
• Current Close Price, Daily Return %, Annualized Volatility & Cumulative Return KPIs
• Candlestick Chart with MA7 & MA30 moving average signals
• Volume Analysis with buy/sell color coding
• Risk vs Reward scatter plot (Daily Return % vs Volatility)
• Cumulative Return Trend with minimum reference line
• Stock Activity Ledger with Volume Change %
• Interactive Filter Panel — filter by Market, Stock & Date Range dynamically

💡 Key learning: Building an end-to-end data pipeline from API → Python transformation → Tableau visualization taught me how real-world BI dashboards actually work — not just the visuals, but the entire data engineering behind them.
